Projects

California House Price Prediction

I created a hyperparamater tuned Random Forest model that was trained on over 20,000 houses to predict housing price in California. I developed a website using Flask to allow anyone with an internet connection to input their housing data and recieve the model's predictions.

X (Twitter) Hate Speech Detection

I used Tensorflow to make a neural network to detect whether X posts should be removed for containing hate speech. I executed a sentiment analysis on over 32,000 posts. Finally, I contrasted this deep learning approach with a logistic regression model.

R2 Visualizer

Uno
